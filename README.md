## 使用mui.js 
## 使用mui.css 
## Bootstrap.js Bootstrap.css

## 使用bigmap地图的api 
bigmap地址  http://www.bigemap.com/offlinemaps/demo/Vuedemo.html#Vuedemo


项目下载和运行
----

- 拉取项目代码
```bash
git clone https://github.com/FEDwxp/webMap.git
cd webMap
```

- 安装依赖
```
yarn install
```

- 开发模式运行
```
gulp
```

- 编译项目
```
gulp build
```
